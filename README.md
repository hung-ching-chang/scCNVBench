# Benchmarking scRNA-seq Copy Number Inference: A Comprehensive Evaluation and Practitioner’s Guide

This repository contains the script used in our benchmarking study for CNV inference. In [our study](https), we benchmark 12 CNV inference methods across 28 real datasets (>100,000 cells) and diverse synthetic datasets.
